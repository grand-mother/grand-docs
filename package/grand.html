API documentation¶

Note

If not stated explicitly, all examples assume that the grand package and astropy.units have been imported as below.

>>> from grand import *
>>> import astropy.units as u

Interfaces¶

The grand package exposes the following interface modules. Most of those are wrappers, E.g. for core C libraries used in the simulation.

  • geomagnet — Geomagnetic helpers
  • io — Utilities for reading or writing data
  • simulation — Simulation helpers
  • store — Interface to remote data storage
  • topography — Topography helpers

Extensions¶

In addition, the grand package extends the astropy package as described below.
